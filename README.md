# 🤖 WhatsApp Auto Reply Bot

Bot WhatsApp otomatis untuk merespon pesan "P" dengan pilihan menu bantuan.

### 🚀 Deploy ke Heroku
Klik tombol di bawah ini untuk menjalankan bot secara gratis:

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/rizamaplikasi/panel)

---

### 📝 Cara Aktivasi Setelah Klik Deploy:
1. Beri nama aplikasi kamu di Heroku (bebas, huruf kecil semua).
2. Klik **Deploy App** dan tunggu sampai selesai.
3. Jika sudah selesai, klik **Manage App**.
4. Klik tombol **More** (pojok kanan atas) > pilih **View Logs**.
5. Tunggu sampai **QR Code** muncul di log tersebut.
6. Buka WhatsApp di HP kamu > Perangkat Tertaut > **Scan QR** tersebut.
7. **Selesai!** Coba chat nomor WA kamu dengan pesan "p".
